## Transparency (Opacity)

Transparency controls how see-through an element is:

| Opacity | Effect |
|---------|--------|
| 100% | Fully solid, no transparency |
| 50% | Half transparent |
| 0% | Completely invisible |

### When to Use Transparency

* **Overlay panels** - Let background show through
* **Subtle backgrounds** - Soften strong colors
* **Fade effects** - Create depth layers
* **Hide elements** - Temporarily remove without deleting

### Setting Transparency

1. Find the element's opacity or transparency setting
2. Adjust the slider or enter a percentage
3. Preview the effect
4. Save

## Color Overlays

Color overlays tint an image with a solid color:

* **Original image** stays underneath
* **Color layer** is applied on top
* **Blend** creates a unified look

Use overlays to:
* Match images to your brand color
* Darken images for better text readability
* Create a cohesive color scheme

## Gradient Backgrounds

Some elements support gradient backgrounds (two or more colors blending together):

| Gradient Type | Description |
|---------------|-------------|
| Linear | Colors blend in a straight line |
| Radial | Colors blend from center outward |

**If available in your dashboard:**
1. Select the start color
2. Select the end color
3. Choose direction or style
4. Preview and save

## Effect Examples

### Darkening for Readability

To make text easier to read on a busy background:
1. Add a dark color overlay (black or dark blue)
2. Set opacity to 30-50%
3. Place text on top

### Brand Tinting

To make any image match your brand:
1. Add your brand color as overlay
2. Set opacity to 20-40%
3. Original image shows through in your brand color

### Subtle Depth

To add sophistication:
1. Use a subtle gradient (white to light gray)
2. Apply as background
3. Creates sense of depth

## Best Practices

* **Subtle is better** - Heavy effects can look amateur
* **Maintain readability** - Content should always be clear
* **Be consistent** - Use similar effects throughout
* **Test on devices** - Effects may look different on mobile

## When NOT to Use Effects

* When it makes text hard to read
* When it distracts from the CTA
* When it clashes with brand guidelines
* When simpler looks better
