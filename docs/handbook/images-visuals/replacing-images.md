# Replacing Images

> **FREE** - All image changes are included at no extra cost

Learn how to upload and replace images throughout your playable ad.

## What You Can Replace

Image elements in the dashboard include:

* Logos and brand marks
* Icons and symbols
* Background images
* Character sprites
* UI elements (buttons, frames, etc.)

## How to Replace an Image

1. **Find the image element** in the dashboard
2. **Click "Upload" or "Replace"**
3. **Select your image file** from your computer
4. **Wait for upload** to complete
5. **Preview** how it looks
6. **Save** your changes

## Supported Formats

| Format | Best For | Transparency |
|--------|----------|--------------|
| PNG | Logos, icons, UI elements | Yes |
| JPG/JPEG | Photos, backgrounds | No |

**Recommendation:** Use PNG for anything that needs a transparent background (logos, icons). Use JPG for large photos or backgrounds.

## Image Size Guidelines

| Element Type | Recommended Size |
|--------------|-----------------|
| Logo | 512 x 512 px |
| Icon | 256 x 256 px |
| Background | 1920 x 1080 px |
| Character | Original aspect ratio |

**Maximum file size:** 2MB per image. Larger files may slow loading.

## Preparing Your Images

Before uploading:

1. **Resize to recommended dimensions** - Oversized images waste bandwidth
2. **Optimize for web** - Compress without losing quality
3. **Check transparency** - Ensure transparent areas are truly transparent
4. **Use the right format** - PNG for transparency, JPG for photos

## Show/Hide Images

Some images have a visibility toggle:

* **Show** - Image appears in the ad
* **Hide** - Image is invisible (useful for A/B testing)

## Color Overlay

Some images support color tinting:

* Add a color layer on top of the image
* Useful for matching brand colors
* Original image stays intact

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Upload fails | Check file size (under 2MB) |
| Image looks blurry | Upload higher resolution |
| Wrong aspect ratio | Resize before uploading |
| Transparency not working | Save as PNG, not JPG |
