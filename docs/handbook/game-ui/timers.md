# Timers

> **FREE** - All timer styling is included at no extra cost

Customize how countdown timers and clocks appear in your playable ad.

## What Are Timers?

Timers show:

* Countdown to end of game
* Time remaining for actions
* Elapsed play time
* Cool-down periods

## What You Can Customize

| Property | What It Does |
|----------|--------------|
| Text Color | Color of the numbers |
| Text Size | How large the numbers appear |
| Position | Where the timer displays |
| Background | Panel behind the timer (if any) |

## Customizing Timers

1. Find timer settings in the dashboard
2. Adjust text color to match your brand
3. Modify size if needed
4. Preview the timer counting down
5. Save

## Timer Styling Tips

### Text Color

| Background | Recommended Text |
|------------|------------------|
| Dark | White or light color |
| Light | Black or dark color |
| Busy | White with dark outline |

### Text Size

* Large enough to read at a glance
* Not so large it distracts from gameplay
* Proportional to other UI elements

### Urgency Effects

Some timers change color when time is low:

| Time Remaining | Color |
|----------------|-------|
| Plenty of time | Normal (white/brand color) |
| Getting low | Yellow/orange |
| Critical | Red |

If available, these can be customized too.

## Why Can't I Change the Time?

Timer numbers are controlled by the game:

| You CAN change | You CANNOT change |
|----------------|-------------------|
| Text colors | Starting time |
| Text size | Countdown speed |
| Position | Timer logic |

For timing adjustments, see [Timing & Speed](../game-settings/timing-speed.md).

## Common Timer Styles

| Style | Color | Size | Effect |
|-------|-------|------|--------|
| Minimal | White | Medium | Clean look |
| Urgent | Red when low | Large | Creates pressure |
| Brand | Brand color | Medium | Consistent branding |
| Subtle | Gray | Small | Doesn't distract |
