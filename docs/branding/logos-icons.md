# Logos & Icons

Replace the default logo with your brand's logo.

## Uploading Your Logo

<figure><img src="../.gitbook/assets/logo-upload-panel.png" alt="Logo Upload Panel"><figcaption>Click Upload to select your logo file</figcaption></figure>

1. Find **Logos** in the Images section
2. Click **Upload** or **Replace**
3. Select your logo file
4. Preview and Save

## Logo Requirements

| Requirement | Specification |
|-------------|---------------|
| Size | 512 x 512 pixels |
| Format | PNG (transparent background) |
| File size | Under 500KB |

<figure><img src="../.gitbook/assets/logo-requirements.png" alt="Logo Requirements"><figcaption>Left: Correct (transparent PNG). Right: Incorrect (white background)</figcaption></figure>

## Preparing Your Logo

### Remove Background

If your logo has a white or colored background:
1. Use a design tool to remove the background
2. Export as PNG (preserves transparency)

### Size It Right

| Issue | Solution |
|-------|----------|
| Too small = pixelated | Use 512x512 minimum |
| Too large = slow | Keep under 500KB |

## App Icons

| Property | Specification |
|----------|---------------|
| Size | 256 x 256 pixels |
| Format | PNG |
| Shape | Square (corners rounded automatically) |

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| White box around logo | Export with transparent background |
| Logo too small/blurry | Use 512x512 source minimum |
| Logo hard to see | Adjust background contrast |
