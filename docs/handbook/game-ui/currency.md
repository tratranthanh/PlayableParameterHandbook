# Currency

> **FREE** - All currency display styling is included at no extra cost

Customize how coins, gems, and other in-game currencies appear.

## What Are Currency Displays?

Currency displays show:

* Coins collected
* Gems or diamonds
* Stars earned
* Any collectible resource

## What You Can Customize

| Property | What It Does |
|----------|--------------|
| Icon | The coin/gem image |
| Icon Color | Tint applied to icon |
| Text Color | Number display color |
| Text Size | How large numbers appear |
| Position | Where currency shows |

## Customizing Currency Displays

1. Find currency settings in the dashboard
2. Upload your custom icon (if replacing)
3. Adjust text colors and size
4. Preview with sample amounts
5. Save

## Currency Icon Tips

### Replacing Icons

If you're replacing the default icon:

| Property | Recommendation |
|----------|----------------|
| Size | 64x64 or 128x128 pixels |
| Format | PNG with transparency |
| Style | Clear, recognizable at small size |
| Color | Distinctive (gold, blue, green) |

### Icon Tinting

Instead of replacing, you can tint existing icons:

* Gold tint for coins
* Blue for gems
* Green for special currency
* Brand color for custom feel

## Text Styling

Currency amounts should be:

* **Readable** - Clear at a glance
* **Visible** - Contrasts with background
* **Consistent** - Matches other UI numbers

### Color Recommendations

| Currency Type | Icon | Text |
|---------------|------|------|
| Coins | Gold/yellow | White or gold |
| Gems | Blue/purple | White or blue |
| Stars | Yellow/gold | White or yellow |
| Custom | Brand color | White or brand |

## Common Currency Styles

| Style | Description |
|-------|-------------|
| Classic | Gold coin, white text |
| Premium | Blue gem, silver text |
| Casual | Colorful star, fun colors |
| Brand | Custom icon, brand colors |

## Why Can't I Change Currency Amounts?

Currency values are game-controlled:

| You CAN change | You CANNOT change |
|----------------|-------------------|
| Icon appearance | Starting amount |
| Number styling | Earn rates |
| Display position | Currency logic |
