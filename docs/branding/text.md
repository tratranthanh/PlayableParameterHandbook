# Text & Headlines

> **FREE** - All text changes are included at no extra cost

Learn how to find and edit text elements throughout your playable ad.

## Finding Text Elements

Text elements in the dashboard include:

* Game titles and headlines
* Button labels (like "Play Now" or "Download")
* Instructions and tutorial text
* End card messages
* UI labels

In the dashboard, text elements are grouped by screen or feature.

## How to Edit Text

1. **Find the text element** in the dashboard
2. **Click the text field** to select it
3. **Delete or modify** the existing text
4. **Type your new text**
5. **Preview** your changes
6. **Save** when ready

## Character Limits

Some text fields have character limits to ensure the text fits properly:

| Element Type | Typical Limit |
|--------------|---------------|
| Headlines | 30-50 characters |
| Button labels | 15-25 characters |
| Short instructions | 100 characters |
| Longer text | 200+ characters |

If your text is too long, you'll see a warning. Try shortening it or using abbreviations.

## Special Characters

The dashboard supports:

* **Letters** - A-Z, a-z, and international characters
* **Numbers** - 0-9
* **Common symbols** - ! ? . , ' " - + = $ %
* **Emojis** - Most common emojis work, but test to be sure

## Text Styles

Depending on your ad, you may be able to customize:

| Property | What It Does |
|----------|--------------|
| Font size | How large the text appears |
| Text color | Color of the text |
| Outline color | Border around text |
| Shadow | Drop shadow behind text |

## Tips for Great Text

* **Keep it short** - Shorter text is easier to read on mobile
* **Use action words** - "Play", "Download", "Try", "Get"
* **Be clear** - Avoid jargon or complex language
* **Match your brand voice** - Casual or professional, stay consistent

## Examples

| Original | Better |
|----------|--------|
| "Click here to begin playing" | "Play Now" |
| "Download our application" | "Get the App" |
| "Congratulations you won!" | "You Won!" |

## Multiple Languages

If your ad supports multiple languages:

1. Find the language selector in the dashboard
2. Switch to the target language
3. Edit text for that language
4. Repeat for all supported languages
5. Test each language version

See [Reference > Limits](../reference/limits.md) for supported languages.
