# Sliders

> **FREE** - All slider customizations are included at no extra cost

Sliders let users adjust values like volume, brightness, or game settings.

## What Are Sliders?

Sliders are draggable controls for adjusting values within a range:

* Volume controls
* Brightness settings
* Progress indicators
* Value adjustments

## What You Can Customize

| Property | What It Does |
|----------|--------------|
| Track Color | The background bar |
| Fill Color | The filled portion |
| Handle Color | The draggable knob |
| Handle Image | Custom handle appearance |

## Customizing Sliders

1. Find the slider settings in the dashboard
2. Select the component to customize (track, fill, handle)
3. Choose your color or upload an image
4. Preview the slider in action
5. Save

## Slider Anatomy

```
[Handle]
   |
===|=================  <- Track
===                    <- Fill (colored portion showing current value)
```

### Each Part

* **Track** - The full length bar (usually gray or subtle)
* **Fill** - Shows the current value (usually colored)
* **Handle** - The draggable part (knob or thumb)

## Best Practices

### Colors

| Component | Recommended |
|-----------|-------------|
| Track | Light gray or subtle color |
| Fill | Brand accent color |
| Handle | White or high contrast |

### Visual Clarity

* Fill color should clearly show the current value
* Handle should be easy to see and grab
* Track should provide context for the full range

## Common Slider Uses

| Slider | Purpose |
|--------|---------|
| Volume | Adjust sound level |
| Music | Adjust background music level |
| Brightness | Adjust screen brightness |
| Sensitivity | Adjust control responsiveness |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Fill color invisible | Check contrast with track |
| Handle hard to see | Use contrasting color |
| Slider looks generic | Apply brand colors |
