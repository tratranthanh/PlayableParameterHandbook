# Multiple Languages

> **FREE** - Language customization is included at no extra cost

Learn how to handle text for different languages and regions.

## Language Considerations

When customizing text for different markets, keep these factors in mind:

### Text Length

Different languages have different lengths for the same message:

| Language | "Download Now" | Length |
|----------|----------------|--------|
| English | Download Now | 12 chars |
| German | Jetzt herunterladen | 19 chars |
| Spanish | Descargar ahora | 15 chars |
| French | Telecharger maintenant | 23 chars |
| Japanese | ダウンロード | 6 chars |

Plan for the longest version to avoid text overflow.

## Tips for Multilingual Text

### Keep It Short

* Use the shortest version possible
* Avoid long phrases when a word will do
* Test how text looks in the preview

### Check Character Support

Most common characters are supported:

* **Latin alphabets** - English, Spanish, French, German, etc.
* **Asian characters** - Japanese, Chinese, Korean
* **Cyrillic** - Russian, Ukrainian, etc.
* **Arabic script** - Arabic, Persian, Urdu

### Right-to-Left Languages

Some languages (Arabic, Hebrew) read right-to-left. Notes:

* Text alignment may need adjustment
* Preview carefully to ensure proper display
* Contact support if you have issues with RTL text

## Best Practices

1. **Test thoroughly** - Preview all text in your target language
2. **Leave room** - Design layouts that accommodate longer text
3. **Use simple words** - Easier to translate and fit
4. **Check fonts** - Some fonts don't support all characters

## Common Issues

| Problem | Solution |
|---------|----------|
| Text too long | Shorten the message |
| Characters not showing | Check font support, contact support |
| Text overlapping | Reduce font size |
| Alignment issues | Preview and adjust spacing |

## Working with Translators

When providing text to translators:

* Give them context about where the text appears
* Specify character limits
* Ask for the shortest natural translation
* Provide screenshots if possible
