# Toggles & Switches

> **FREE** - All toggle customizations are included at no extra cost

Toggles are on/off switches used for settings like sound, notifications, or feature controls.

## What Are Toggles?

Toggles let users switch something on or off:

* Sound on/off
* Music on/off
* Notifications on/off
* Feature enables/disables

## What You Can Customize

| Property | What It Does |
|----------|--------------|
| On Color | Color when toggle is "on" |
| Off Color | Color when toggle is "off" |
| Handle Color | The sliding button part |
| Track Color | The background track |

## Customizing Toggles

1. Find the toggle settings in the dashboard
2. Select which color property to change
3. Choose your color (use brand colors for consistency)
4. Preview both states (on and off)
5. Save

## Toggle Best Practices

### Colors

| State | Recommended |
|-------|-------------|
| On | Brand accent color (green often works) |
| Off | Gray or muted color |
| Handle | White or contrasting color |

### Visual Clarity

* **Clear difference** between on and off states
* **Consistent** with your brand colors
* **Visible** against the background

## Common Toggle Uses

| Toggle | Purpose |
|--------|---------|
| Sound | Mute/unmute game audio |
| Music | Mute/unmute background music |
| Vibration | Enable/disable haptic feedback |
| Notifications | Allow/block alerts |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| States look too similar | Use contrasting on/off colors |
| Toggle hard to see | Check background contrast |
| Wrong default state | Check settings configuration |
