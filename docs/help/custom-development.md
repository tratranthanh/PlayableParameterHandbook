# Custom Development

Need something not in your dashboard? We can build it.

## Default vs Custom

| Default Parameters | Custom Development |
|-------------------|-------------------|
| In your dashboard | Not in dashboard |
| Built-in, apply automatically | Requires development |
| No order needed | Order required |

## Examples

| Default (no order needed) | Custom (order required) |
|---------------------------|------------------------|
| Change button color | Add new button type |
| Edit existing text | Add new text fields |
| Replace images | Generate map from image |
| Adjust timer display | Add new timer logic |
| Style HP bar | Add health mechanics |
| Adjust animation presets | Create new animations |

---

## How to Request

Tell us:
1. **What** — What should happen? How should it look?
2. **Why** — What goal does this achieve?
3. **References** — Screenshots, examples, mockups
4. **Timeline** — When do you need it?

## What Happens Next

1. We receive your request (within 1 business day)
2. We assess the work
3. You receive a quote with scope, cost, and timeline
4. You decide (no obligation)

---

## Pricing

Custom development is quoted per project. Your quote includes:

| Item | Description |
|------|-------------|
| Scope | What will be built |
| Cost | Total price |
| Timeline | Delivery time |

> 💡 **Default parameters** (in your dashboard) have no additional cost — adjust freely.
