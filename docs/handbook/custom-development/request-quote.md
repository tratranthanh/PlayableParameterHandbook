# Request a Quote

Ready to explore custom development? Here's how to get started.

## How to Request Custom Work

### Step 1: Describe What You Want

Tell us:

* **What should happen?** - Describe the feature or behavior
* **When should it appear?** - At what point in the ad?
* **How should it look?** - Visual style, colors, design

### Step 2: Explain Why You Need It

Help us understand:

* **What goal does this achieve?** - Marketing objective, conversion goal
* **Why can't dashboard options work?** - What's missing?
* **How important is this?** - Priority level

### Step 3: Share Reference Examples

The more context, the better:

* Screenshots or videos of similar features
* Links to ads with features you like
* Sketches or mockups
* Competitor examples

### Step 4: Timeline

Let us know:

* When do you need this?
* Is there flexibility?
* Any hard deadlines?

## What to Include in Your Request

```
Feature Name: [Brief title]

Description:
[What the feature should do]

Visual Reference:
[Links to examples, screenshots, or mockups]

Why It's Needed:
[Business goal or user experience improvement]

Timeline:
[Deadline or preferred delivery date]

Priority:
[High / Medium / Low]
```

## What Happens Next

1. **We receive your request** - Usually within 1 business day
2. **We assess the work** - Determine scope and complexity
3. **You receive a quote** - With estimated cost and timeline
4. **You decide** - No obligation to proceed

## Quote Details

Your quote will include:

| Item | Description |
|------|-------------|
| Scope | Exactly what will be built |
| Cost | Total price for the work |
| Timeline | How long it will take |
| Deliverables | What you'll receive |

## Tips for Faster Quotes

* **Be specific** - Vague requests take longer to scope
* **Include visuals** - Screenshots and mockups help a lot
* **Prioritize** - If requesting multiple features, rank them
* **Be responsive** - Quick answers speed up the process
