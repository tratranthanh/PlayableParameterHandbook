## What to Include in Your Request

Help us help you faster by including:

### For Technical Issues

1. **What you were trying to do**
   * Which feature were you using?
   * What setting were you changing?

2. **What happened**
   * Error messages (copy the exact text)
   * Unexpected behavior

3. **Screenshots**
   * Screen showing the issue
   * Any error messages

4. **Your environment**
   * Browser and version
   * Device (desktop/mobile)

### For Feature Questions

1. **What you want to achieve**
   * Describe your goal
   * Include examples if possible

2. **What you've tried**
   * Settings you've checked
   * What didn't work

## Request Template

```
Subject: [Brief description of issue]

Issue Type: [Bug / Question / Feature Request / Account Issue]

Description:
[Detailed explanation of what's happening]

Steps to Reproduce:
1. [First step]
2. [Second step]
3. [What happened]

Expected Result:
[What you expected to happen]

Actual Result:
[What actually happened]

Browser/Device:
[e.g., Chrome 100 on Windows 10]

Screenshots:
[Attach if applicable]
```
