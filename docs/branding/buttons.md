# Buttons

The most important interactive elements in your ad.

## Button Parameters

If these appear in your dashboard:

| Property | What It Does |
|----------|--------------|
| Button Background | The main button color or image |
| Button Text | The label (like "Play Now") |
| Text Color | Color of the button text |
| Text Size | How large the text appears |
| Button Color | Background color of the button |

## Customizing Your CTA Button

The Call-to-Action (CTA) button is the most important element in your ad. It's what you want users to tap!

### To customize:

1. Find the CTA button settings in the dashboard
2. Update the text to your desired message
3. Change colors to match your brand
4. Adjust text size if needed
5. Preview on different screen sizes
6. Save

## CTA Button Best Practices

### Text

| Do | Don't |
|----|-------|
| "Play Now" | "Click here" |
| "Download Free" | "Submit" |
| "Try It!" | "Continue" |
| "Get the App" | "OK" |

Use action words that create excitement!

### Colors

* **Stand out** - CTA should be the most visible element
* **Brand aligned** - Use your accent color
* **High contrast** - Text must be readable
* **Consistent** - Same style on all CTA buttons

### Size

* Large enough to tap easily on mobile (minimum 44x44 pixels)
* Not so large it overwhelms other content
* Proportional to the screen

## Button States

Some buttons have different states:

| State | When It Shows |
|-------|---------------|
| Normal | Default appearance |
| Pressed | When user is tapping |
| Disabled | When action isn't available |

If available, you can customize colors for each state.

## Multiple Buttons

If your ad has multiple buttons:

* **Primary CTA** - Most prominent, brightest color
* **Secondary buttons** - Less prominent, subtler style
* **Consistent styling** - Similar buttons should look similar

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Text doesn't fit | Shorten the text or reduce font size |
| Button hard to see | Increase contrast with background |
| Text not readable | Check text color vs button color |
| Wrong color | Double-check hex codes |
