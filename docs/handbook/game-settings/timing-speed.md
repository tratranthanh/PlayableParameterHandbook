# Timing & Speed

> **FREE** - All timing adjustments are included at no extra cost
> 
> *Note: Some gameplay changes may require brief review before going live*

Control how fast or slow different elements of your playable ad behave.

## What You Can Adjust

| Setting | What It Does |
|---------|--------------|
| Timer Duration | How long users have to play |
| Animation Speed | How fast animations play |
| Movement Speed | How fast characters/objects move |
| Transition Time | How long screens fade in/out |
| Delay Settings | Pauses between events |

## Adjusting Timing

1. Find timing settings in the dashboard
2. Review current values (usually in seconds)
3. Adjust to your preferred timing
4. Preview the changes
5. Save (may require review)

## Timing Guidelines

### Timer Duration

How long should the playable ad last?

| Duration | Best For |
|----------|----------|
| 15-30 seconds | Quick previews |
| 30-45 seconds | Standard playables |
| 45-60 seconds | Deeper gameplay |
| 60+ seconds | Complex games |

**Tip:** Most successful playable ads are 30-45 seconds.

### Animation Speed

| Speed | Effect |
|-------|--------|
| Slower | More dramatic, easier to follow |
| Normal | Matches full game |
| Faster | More energetic, covers more content |

### Movement Speed

| Speed | Effect |
|-------|--------|
| Slower | Easier to control, more relaxed |
| Normal | Authentic game feel |
| Faster | More challenging, exciting |

## Common Timing Adjustments

### Shorten the Experience

To make the ad quicker:
* Reduce timer duration
* Speed up animations
* Reduce delays between screens

### Extend the Experience

To let users play longer:
* Increase timer duration
* Add time bonuses
* Slow transitions slightly

### Make It More Accessible

To help users succeed:
* Slower movement speeds
* Longer reaction windows
* More forgiving timing

## Why Review May Be Required

Timing changes affect how the game feels:

| Low-Impact Changes | Higher-Impact Changes |
|-------------------|---------------------|
| Transition speed | Game timer duration |
| Visual animations | Movement speed |
| Fade durations | Gameplay pacing |

Review ensures changes maintain a quality experience.

## Testing Your Changes

After adjusting timing:

1. Play through completely
2. Check if it feels natural
3. Verify it's not too rushed or slow
4. Ensure core gameplay is shown
