# Common Parameters

Parameters vary by playable. Below are common parameters found in most builds.

> **Your dashboard shows the actual parameters available for your specific playable.**

---

## Almost Every Game Has

| Parameter | Type | Description |
|-----------|------|-------------|
| Logo | Image | Your brand logo |
| App Icon | Image | Store icon |
| CTA Button | Text + Color | Call-to-action on end cards |
| Loading Background | Image/Color | Loading screen background |

---

## Common UI Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Primary Color | Color | Main brand color |
| Secondary Color | Color | Supporting color |
| Button Background | Color | CTA button color |
| Button Text | Text | CTA label |
| Headline | Text | Main text on screens |

---

## Common Gameplay Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Timer Duration | Number | Countdown time (seconds) |
| Player HP | Number | Player health value |
| Speed Multiplier | Number | Movement/game speed |
| HP Bar Color | Color | Health bar fill color |
| Score Color | Color | Score display color |

> **Note:** Complex difficulty (AI behavior, spawn logic) needs development. See [Custom Development](../help/custom-development.md).

---

## Common Screen Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Loading Text | Text | "Loading..." message |
| Win Title | Text | Win screen headline |
| Lose Title | Text | Lose screen headline |
| Tutorial Text | Text | Onboarding instructions |

---

## Your Playable May Have More

Each game has unique parameters based on its mechanics:

* **Puzzle games** — grid size, match count, piece colors
* **Action games** — spawn rate, enemy count, weapon damage
* **Idle games** — multipliers, currency rates, upgrade costs
* **Simulation** — timers, resource limits, unlock thresholds

Check your dashboard for the complete list of parameters available in your build.
