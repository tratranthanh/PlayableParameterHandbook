# Gameplay Values

Adjust point values, rewards, and numeric settings.

## Included vs Custom Development

| Included (Adjust Existing) | Custom Development (New Systems) |
|----------------------------|----------------------------------|
| Score display format | New scoring formulas |
| Currency icon/color | New reward distribution |
| Progress bar style | New combo systems |
| Point/reward values | New calculation logic |
| Toggle visibility | New features |

**Tip:** Adjusting existing values = included. Building new systems = quote required.

---

## UI Config Parameters

If these appear in your dashboard:

| Setting | What It Does |
|---------|--------------|
| Score Display | How score numbers appear |
| Currency Icon | Visual for in-game currency |
| Progress Style | How progress is shown |
| Tutorial Toggle | Show/hide tutorial overlay |

---

## Custom Development (Game Logic)

| Request | What's Involved |
|---------|-----------------|
| Point formulas | Calculation code |
| Combo systems | Streak tracking |
| Reward amounts | Game balance work |
| Win conditions | Game flow changes |

---

## Feature Toggles

Some games have visibility toggles:

| Toggle | What It Controls |
|--------|------------------|
| Tutorial | Show/hide tutorial overlay |
| Power-ups | Show/hide ability icons |

> **Note:** Haptics and native mobile features are not supported in playable ads on ad networks.

---

## Best Practices for Playable Ads

| Do | Avoid |
|----|-------|
| Show clear progress | Confusing point systems |
| Achievable goals | Impossible targets |
| Rewarding finish | Frustrating endings |
