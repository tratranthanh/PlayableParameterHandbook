# Colors

Customize colors using hex codes or the color picker.

## Using the Color Picker

<figure><img src="../.gitbook/assets/color-picker.png" alt="Color Picker Interface"><figcaption>Click the color swatch to open the picker</figcaption></figure>

1. Select an element with a color option
2. Click the color swatch
3. Choose visually or enter hex code
4. Preview and Save

## Hex Code Format

| Format | Example |
|--------|---------|
| Standard | #FF5722 |
| With transparency | #FF572280 |

<figure><img src="../.gitbook/assets/color-hex-input.png" alt="Hex Code Input"><figcaption>Enter your exact brand color hex code</figcaption></figure>

## Brand Color Examples

| Style | Primary | Secondary | Accent |
|-------|---------|-----------|--------|
| Blue | #0066CC | #003366 | #FF9900 |
| Green | #00C853 | #1B5E20 | #FFFFFF |
| Red | #E53935 | #B71C1C | #FFC107 |

## Color Parameters

| Element | Where |
|---------|-------|
| Button backgrounds | Images > Buttons |
| Text colors | Images > Text |
| UI elements | Gameplay > UI |

## Tips

* **Use contrast** — Text must be readable
* **Be consistent** — Same colors on similar elements
* **Test on device** — Colors may vary on mobile
