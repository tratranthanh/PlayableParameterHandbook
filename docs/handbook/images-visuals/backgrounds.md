# Backgrounds

> **FREE** - All background changes are included at no extra cost

Transform the look of your playable ad with custom background images.

## Background Types

Different screens may have different background options:

| Screen | Background Type |
|--------|-----------------|
| Main gameplay | Full-screen image |
| Loading screen | Solid color or image |
| End cards | Image behind content |
| UI panels | Decorative frames |

## Changing Backgrounds

1. Find the background setting in the dashboard
2. Upload your new background image
3. Preview how it looks with other elements
4. Adjust if needed
5. Save

## Background Requirements

| Property | Recommendation |
|----------|----------------|
| Size | 1920 x 1080 pixels (landscape) or 1080 x 1920 (portrait) |
| Format | JPG for photos, PNG for graphics |
| File size | Under 1MB ideal, max 2MB |
| Resolution | 72-150 DPI for web |

## Background Tips

### Match the Orientation

* **Portrait mode** - 1080 x 1920 px (tall)
* **Landscape mode** - 1920 x 1080 px (wide)

Check which orientation your ad uses before designing backgrounds.

### Consider Content Overlap

UI elements will appear on top of the background:

* Keep the center relatively simple
* Avoid important details where buttons will be
* Use darker or lighter areas where text appears
* Test readability with actual content

### Create Visual Hierarchy

* **Darker areas** behind light text
* **Lighter areas** behind dark text
* **Less busy areas** behind important buttons

## Stretched vs. Tiled

Some backgrounds offer display options:

| Option | Best For |
|--------|----------|
| Stretched | Photos, gradients |
| Tiled | Repeating patterns |
| Centered | Logos, focal images |

## Background Ideas

* **Brand gradient** - Your brand colors fading together
* **Product photo** - Your app or game in action
* **Abstract pattern** - Shapes in brand colors
* **Subtle texture** - Adds depth without distraction
* **Solid color** - Clean and professional

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Background looks stretched | Use correct aspect ratio |
| Text hard to read | Darken/lighten background, add text shadow |
| File too large | Compress or reduce dimensions |
| Image looks pixelated | Upload higher resolution source |
