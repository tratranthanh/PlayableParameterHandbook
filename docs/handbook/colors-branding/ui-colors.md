# UI Colors

> **FREE** - All UI color changes are included at no extra cost

Learn how to change colors on buttons, text, backgrounds, and other UI elements.

## What UI Colors Can You Change?

| Element | Color Options |
|---------|---------------|
| Buttons | Background color, text color, border |
| Text | Text color, outline color |
| Backgrounds | Solid color, overlay tint |
| Bars (HP, progress) | Fill color, background color |
| Frames | Border color, fill color |

## Changing Button Colors

Buttons often have multiple color options:

1. **Background** - The button's main color
2. **Text** - The label text color
3. **Border** - The outline color (if visible)

To update:
1. Find the button in the dashboard
2. Click the color option you want to change
3. Select your color or enter a hex code
4. Preview and save

## Changing Text Colors

Text color options include:

* **Main color** - The fill color of the text
* **Outline color** - Border around letters (for visibility)

Best practice: Choose colors that contrast with the background.

## Background Color vs. Image

Some elements let you choose between:

| Option | When to Use |
|--------|-------------|
| Solid color | Clean look, fast loading, easy branding |
| Image | More visual interest, specific design needs |
| Color overlay | Tint an image with your brand color |

## Color Contrast Guide

For readable text, ensure good contrast:

| Background | Recommended Text |
|------------|-----------------|
| Dark (black, navy) | White, light colors |
| Light (white, cream) | Black, dark colors |
| Medium (gray) | White or black with outline |
| Colorful | White or black (test both) |

### Accessibility Tip

Make sure your text colors have enough contrast against backgrounds. This helps everyone read your ad clearly, including users with vision differences.

**Quick test:** If you squint and the text disappears into the background, you need more contrast.

## Applying Colors Consistently

For a polished look, use a consistent color system:

| Purpose | Your Color |
|---------|------------|
| Primary actions (main CTA) | Brand primary |
| Secondary actions | Brand secondary |
| Warnings/alerts | Orange or red |
| Success/positive | Green |
| Text on light | Dark gray or black |
| Text on dark | White or light gray |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Color doesn't match | Check hex code is correct |
| Text hard to read | Increase contrast |
| Colors look different | Display settings vary; use hex codes |
| Button color missing | Check both background and border |
