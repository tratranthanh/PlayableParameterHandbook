# FAQ

Find quick answers to frequently asked questions.

## Dashboard Parameters

### What can I adjust?

Parameters in your dashboard are default (built-in) — adjust and apply automatically, no order needed.

### What's included in my dashboard?

Each playable has different parameters based on its build. Common types:
* Images (logos, icons, backgrounds)
* Text (headlines, CTA, instructions)
* Colors (buttons, UI, backgrounds)
* Audio (music, sound effects)
* Animation presets (fade, bounce, slide)

### What if I need something not in my dashboard?

We can build it through custom development. Common requests:
* Generate map from image
* Create new animations
* Add game mechanics
* New screen types

See [Custom Development](custom-development.md) to request.

## Making Changes

### Will my changes go live immediately?

Most visual changes (colors, images, text) apply instantly to your preview. Some gameplay changes may need a brief review before going live.

### Can I undo my changes?

Yes! The dashboard keeps your original settings. You can:
* Revert individual changes
* Reset to the original version
* Preview before saving

### Can I test before going live?

Absolutely! Always use the Preview feature to see your changes before making them live. Test as many times as you want.

### What if I make a mistake?

Don't worry! You can:
* Use Preview to check changes first
* Revert changes you don't like
* Reset to original settings
* Contact support for help

## Technical Questions

### What image formats work?

* **PNG** - Best for logos, icons, transparency
* **JPG/JPEG** - Good for photos and backgrounds

See [File Formats](../reference/file-formats.md) for detailed specs.

### What's the maximum file size?

Images should be under 2MB. Larger files may slow down loading.

See [Limits & Constraints](../reference/limits.md) for all limits.

### What screen sizes are supported?

Your playable ad automatically adapts to different screen sizes. Design for standard mobile dimensions (1080x1920 portrait or 1920x1080 landscape).

### Can I use any font?

The dashboard uses web-safe fonts that work across all devices. Custom fonts may be available through custom development.

## Account Questions

### How do I get a dashboard account?

Contact your account manager. They'll set up your access and provide login credentials.

### Can multiple people access the same dashboard?

Yes, your team can have multiple accounts. Contact your account manager to set up additional users.

### I forgot my password. What do I do?

Use the "Forgot Password" link on the login page. You'll receive an email to reset your password.

## Feature Questions

### Can I change [specific feature]?

Check your dashboard for available parameters. If it's visible, it's ready to adjust. For features not in your dashboard, see [Custom Development](custom-development.md).

### Why can't I edit some things?

Some elements are controlled by game logic (like actual scores or timer numbers). You can style how they look, but the values are determined by gameplay.

### Can I add my own features?

New features not in the dashboard require custom development. See [Custom Development](custom-development.md) for more information.
