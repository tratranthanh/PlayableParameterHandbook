# Text Styles

> **FREE** - All text styling is included at no extra cost

Make your text stand out with colors, sizes, and formatting options.

## Available Style Options

| Option | What It Does |
|--------|--------------|
| Color | Change text color to any hex code |
| Size | Make text larger or smaller |
| Bold | Make text thicker and heavier |
| Italic | Slant text for emphasis |
| Underline | Add a line under text |
| Outline | Add a colored border around letters |

## Changing Text Color

1. Select the text element
2. Find the "Color" option
3. Use the color picker OR enter a hex code (like #FF5722)
4. Preview the change
5. Save

### Entering Hex Codes

Hex codes are 6-character color codes starting with #. Examples:

| Color | Hex Code |
|-------|----------|
| Red | #FF0000 |
| Blue | #0000FF |
| Green | #00FF00 |
| Orange | #FF5722 |
| Black | #000000 |
| White | #FFFFFF |

Ask your marketing team for your brand's official hex codes!

## Changing Text Size

Text size is measured in points or pixels:

| Size | Best For |
|------|----------|
| Small (12-16) | Fine print, disclaimers |
| Medium (18-24) | Body text, instructions |
| Large (28-36) | Headlines, important text |
| Extra Large (40+) | Main titles, big CTAs |

## Adding Formatting

### Bold
Makes text **thicker and heavier**. Great for:
* Headlines
* Important words
* Button text

### Italic
Makes text *slanted*. Good for:
* Emphasis
* Quotes
* Special terms

### Underline
Adds a line under text. Use sparingly - often looks like a link.

## Text Outlines

Outlines add a colored border around each letter. Useful when:

* Text is over a busy background
* You want text to "pop"
* Need high contrast

Options:
* **Outline Color** - The border color
* **Outline Width** - How thick the border is

## Best Practices

* **Consistency** - Use the same style for similar elements
* **Readability** - Dark text on light backgrounds (or vice versa)
* **Hierarchy** - Bigger/bolder for important items
* **Brand alignment** - Match your company's typography style
