# Score Displays

> **FREE** - All score styling is included at no extra cost

Customize how scores and points appear in your playable ad.

## What Are Score Displays?

Score displays show:

* Current points
* High scores
* Combo multipliers
* Achievement counts

## What You Can Customize

| Property | What It Does |
|----------|--------------|
| Text Color | Color of score numbers |
| Text Size | How large scores appear |
| Position | Where scores display |
| Effects | Pop/pulse animations (if available) |

## Customizing Score Displays

1. Find score display settings in the dashboard
2. Choose text color (consider background visibility)
3. Adjust size for readability
4. Preview with sample numbers
5. Save

## Score Styling Tips

### Visibility

Scores should be:

* Easy to read quickly
* Visible against the background
* Noticeable but not distracting

### Color Choices

| Purpose | Recommended |
|---------|-------------|
| Main score | White or brand color |
| Combo | Accent color (gold, orange) |
| Bonus | Green or highlight color |

### Size Guidelines

| Type | Size |
|------|------|
| Main score | Large, prominent |
| Secondary stats | Medium |
| Small info | Smaller, subtle |

## Score Effects

Some score displays have special effects:

| Effect | When It Happens |
|--------|-----------------|
| Pop/bounce | Score increases |
| Glow | Big score jump |
| Color change | Milestone reached |

If available in your dashboard, these can add excitement to scoring.

## Why Can't I Change the Points?

Score values are game-controlled:

| You CAN change | You CANNOT change |
|----------------|-------------------|
| Number colors | Point values |
| Number size | Scoring rules |
| Display position | Multipliers |

For point value adjustments, see [Gameplay Values](../game-settings/gameplay-values.md).
